# turkuforms 2024

### Vaadin 23+ implementation for moware dataUx IToolkit interface. 

Moware DataUX is a DSL (Domain Specific Language) to model user interfaces for business applications (<https://www.modellwerkstatt.org/DSL_for_Business_Applications.pdf>). The DSL is part 
of moware werkbank, a DSL stack of 3 languages capable of modelling all aspects of business apps (<https://www.modellwerkstatt.org/werkbank.html>) in jetbrains MPS. 

Turkuforms is a web component based implementation for the dataux runtime. It is fully compatible to the java FX desktop 
implementation. Turkuforms is baed on java 11 and vaadin 23+.
